# Mandelbrot
### How use?

### Install lib
```
# apt-get install libsfml-dev
# wget https://github.com/texus/TGUI/archive/0.8-dev.zip
# unzip 0.8-dev.zip
# cd 0.8-dev.zip
# cmake .
# make -j2
# sudo make install
# sudo ldconfig
```
### Install app
```
# make && ./mandelbrot
```
